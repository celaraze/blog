Ubuntu 22.04 个人安装配置

配置 oh-my-zsh

安装 ZSH：sudo apt install zsh

安装 oh-my-zsh：

sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"

安装 zsh-autosuggestions 自动补全：

git clone https://github.com/zsh-users/zsh-autosuggestions.git $ZSH_CUSTOM/plugins/zsh-autosuggestions

安装 zsh-syntax-highlighting 语法高亮： git clone https://github.com/zsh-users/zsh-syntax-highlighting.git $ZSH_CUSTOM/plugins/zsh-syntax-highlighting

配置 CIFS

sudo apt install cifs-utils

解决 PCIE Bus Error

这个问题大概率是由于 nvme 硬盘引起的，是 Ubuntu 内核问题。

修改 GRUB ，sudo nano /etc/default/grub

修改完成后更新 GRUB，sudo update-grub

然后重启服务器。
